# Sniffer
自写抓包+arp欺骗程序

计算机网络安全试验

## 使用

Include和WpdPack文件夹为winpacp的文件，包括lib等。其中Include为WpdPack文件中的一个子文件夹，此处移出使用。相对于源码略有修改，主要是将<>改为"".

如需使用，请修改pro文件中的lib路径。以前使用了绝对路径。

## 功能
	双击网卡抓包
	单机网卡，点击arp欺骗按钮抓包

## 功能介绍

### arp欺骗

arp欺骗是对网关及目的主机，使用广播地址进行欺骗。

### 包存储

包存储使用了结构体顺序存储

### 过滤器设置

过滤器设置文件保存为set。

### http应用层协议

针对端口是80认为是http协议，其他端口暂未认定
